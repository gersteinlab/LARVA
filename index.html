LARVA

We have developed a computational framework for identifying rare germline variants, named Large-scale Analysis of Recurrent Variants and Annotations (LARVA). Given a set of cancer patient whole genome variant calls, and a set of genome annotations, LARVA will pick out the recurrent variants and recurrently mutated annotations that result from overlapping the variants with the annotations.

Installation

To run LARVA you will need to have the latest version of perl and cpan installed. With this next you will enter the following commands on linux systems. 
$ perl -MCPAN -e shell
$ install Bundle::DBI
$ install DBD::SQLite
next to use LARVA-SAM you will need to have the R programming language installed.
